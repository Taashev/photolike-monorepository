# PhotoLike Frontend

### Технологии при разработке:

- React 18.1
- JavaScript (ES6)
- БЭМ (Nested)
- HTML 5 (Semantic)
- ООП

### Была добавлена следующая функциональность:

- Регистрация
- Авторизация
- Возможность разлогиниться
- Header с личной информацией
- Бургер для мобильной версии

---

### Как запустить frontend

**Установить зависимости**

```sh
npm i
```

**Запустить проект**

```sh
npm run start
```
